# Better Server Website

Welcome to the official website for **Better Server**, a Minecraft server hosted by our community! This site is designed to enhance your experience, connect players, and showcase everything happening on the server.


## Features

- **Home Page**: Introduction to the server, links to other sections, and a summary of what the community offers.
- **Interactive World Map**: View a live map of the Minecraft world (Bluemap integration).
- **Feed**: Live feed where users can post updates and chat. Users can create and delete their own posts (login required).
- **Players Page**: Playerbase/biography page where users can add or update their own profile and description (login required). View other players' bios.
- **Wiki**: A detailed description of all the happenings on the server.
- **Service Status**: See live server status, online/offline state, player count, ping, and joinability. List of currently online players.
- **Account Management**: Sign up, log in, log out, and manage your account. Upload a profile picture. Session-based authentication.
- **Server Status Overlay**: Floating widget showing live server status and player count on every page.
- **Profile Pictures**: Users can upload and display a profile image.
- **Responsive Design**: Mobile-friendly layout and components.


## Tech Stack

- **Frontend**: [React](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Backend**: [Express](https://expressjs.com/)


## Getting Started

1. **Clone the repository**
2. **Install dependencies** for both `client` and `server`:
   ```sh
   cd client
   npm install
   cd ../server
   npm install
   ```
3. **Run the development servers**:
   - For the frontend:
     ```sh
     cd client
     npm run dev
     ```
   - For the backend:
     ```sh
     cd server
     npm run dev
     ```
